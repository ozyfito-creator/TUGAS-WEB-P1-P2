Belajar CSS Box Model

Apa itu Box Model

Setiap elemen HTML dianggap sebagai kotak (box). Box model terdiri dari: content, padding, border, dan margin.

  • Content: Isi sebenarnya dari elemen
  • Padding: Ruang antara content dan border
  • Border: Garis tepi elemen
  • Margin: Ruang di luar border,memisahkan elemen dari elemen lainnya
